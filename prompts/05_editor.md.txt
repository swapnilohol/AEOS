# =============================================================================
# EDITOR ROLE MANAGEMENT
# =============================================================================


Introduce Editor workspace role concept.



Roles:


ADMIN


STUDENT


EDITOR_SESSION



EDITOR_SESSION represents:


A temporary coding workspace created when a student starts solving a problem.



Permissions:


STUDENT:


Can:

- Open assigned problems
- Write code
- Save drafts
- Submit solutions



ADMIN:


Can:

- Monitor sessions
- View submission activity



Student cannot:


- Access other drafts
- Modify hidden test cases
- Access execution environment
